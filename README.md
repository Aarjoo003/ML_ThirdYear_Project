# 🌸 Project PanchTatva: 5 Dimensions of Machine Learning

A unified Machine Learning project integrating **Image, Video, Audio, Text, and Numeric datasets** into one intelligent ecosystem. This project demonstrates real-world applications of ML across multiple data domains.

## 🧠 Project Overview

Project PanchTatva represents five core elements of Machine Learning:

| Domain  | Description |
|--------|------------|
| 👁️ Image | Eye Disease Detection using CNN models |
| 🎥 Video | Photo vs Video classification using metadata |
| 🎧 Audio | Drone sound detection using MFCC + Deep Learning |
| 💬 Text | WhatsApp sentiment analysis using NLP |
| 🔐 Numeric | Zero-Day cyber attack detection |
